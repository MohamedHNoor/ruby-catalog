# ğŸ“– Catalog of my things <a name="about-project"></a>

> This project is a simple Ruby app that allows you to add books, musics and games to a catalog. It also allows you to list all books, music and games. All data is preserved in a json file. The project also includes a schema file with tables of the classes in the app.

## ğŸ’» Getting Started with ruby-catalog <a name="getting-started"></a>
> To get a local copy up and running, follow these steps.

### Prerequisites
In order to run this project you need:
1. Ruby installed on your machine
2. Text editor (VSCode, Sublime, Atom, etc.)

### Setup

Clone this [repository](https://github.com/rbhatt1999/ruby-catalog.git) to your desired folder:

```sh
  cd ruby-catalog
  git clone https://github.com/rbhatt1999/ruby-catalog.git 
```

### Install
Install this project with:
```sh
  cd school-library
```

### Usage
Run this project with:
```sh
  ruby main.rb
```
### Run tests
Run tests with:
```sh
  rspec
  rubocop
```
